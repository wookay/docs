H3.API.h3ToLocalIjk — Function.
h3ToLocalIjk(origin::H3Index, h3::H3Index)::CoordIJK

Produces ijk+ coordinates for an index anchored by an origin. The coordinate space used by this function may have deleted regions or warping due to pentagonal distortion. Coordinates are only comparable if they come from the same origin index. Failure may occur if the index is too far away from the origin or if the index is on the other side of a pentagon.
